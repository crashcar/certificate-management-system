# Vue前端

> vue 2.X

## vue项目运行

```bash
# 安装node.js, 安装cnpm
node -v
npm config set registry https://registry.npm.taobao.org
npm config get registry
npm install -g cnpm  
npm install -g cnpm --registry=http://registry.npm.taobao.org
cnpm install -g yarn
yarn --version
yarn config set registry https://registry.npm.taobao.org


cnpm install webpack -g

# cd web
cnpm install

# 报错解决, 无报错跳过
# cnpm install --save core-js svg-baker-runtime
# cnpm install --save-dev vue-loader@^15.9.3  

# vue2 运行
cnpm run dev

#   App running at:
# - Local:   http://localhost:9528/web/
```
